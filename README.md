# VR-Симулятор_базовых_интерактивных_действий
VR-симулятор базовых интерактивных действий — интерактивная песочница для тестирования механик в VR.  
Особенности:  Физические кнопки и базовые настройки  
Лук с натяжением тетивы и кузнечный молот  
3 вида огнестрельного оружия (разные типы перезарядки)  
Универсальная система хватов  
Отличный фреймворк для VR-разработчиков!
