# VR--

VR-симулятор базовых интерактивных действий — интерактивная песочница для тестирования механик в VR.

## Возможности

- Физические кнопки и базовые интерактивные настройки
- Лук с натяжением тетивы
- Кузнечный молот
- 3 вида огнестрельного оружия с разными типами перезарядки
- Универсальная система хватов

## Для кого проект

Проект подойдет как учебная база и как стартовый фреймворк для VR-разработчиков, которым нужно быстро проверить и доработать игровые механики.
